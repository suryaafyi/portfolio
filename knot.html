Each tool captures a piece.

Research

Understanding how people hold on to what matters


What I set out to learn

  • How people actually remember moments, not just store them
  • Where emotional value gets lost
  • Why existing tools fail during real-life use

What I found

  • People don’t forget. They remember too late
  • The problem isn’t storage, it’s meaningful revisit
  • Privacy changes behavior. People share more in safe spaces
  • Current tools break emotional continuity
The moment it clicked

How might we make digital memories feel like a shared home, not a collection of files?

Solution

A private, emotionally intelligent memory system

System Design

Designed around intent, not features

From notification → to memory creation → to shared interaction

Each step is lightweight, guided, and intentional.

Navigation built for behavior

  • Home → prompts and reminders
  • Timeline → revisit memories
  • Events → upcoming moments
  • Create → capture instantly
  • Profile → relationships and control
Testing

Where the system broke, and why

21.1%
Misclick rate in finding the Family Graph

Setup

10 users · 9 tasks · Interactive prototype

What worked

Every task reached 100% completion.
But that wasn’t the full story.

Why this happened

  • Hidden under Profile
  • Not part of primary navigation
  • No onboarding guidance

What I changed

  • Introduced contextual coach marks
  • Highlighted entry points
  • Guided actions through FAB

Impact

  • Reduced confusion
  • Faster task completion
  • Stronger first-time experience
